Unsupervised Learning Case Studies

A collection of unsupervised machine learning projects developed during my MSc in Artificial Intelligence.

The repository includes practical applications of clustering, dimensionality reduction, topic modelling and anomaly detection. Each case study preserves the original methodology and results while improving the documentation and presentation for portfolio purposes.

Case Studies
1. Machine Failure Sensor Clustering

Comparison of K-Means, DBSCAN and hierarchical clustering using industrial machine sensor data.

Main topics:

Exploratory data analysis
Outlier analysis and feature transformation
Cluster selection and validation
PCA and t-SNE visualisation
Comparison of clustering algorithms
Analysis of the relationship between clusters and machine failures
Review of scientific applications of clustering
